Project Description


Vehicle Collisions can cause injury, disability and even death to the individuals involved. Also, the financial cost of property damages is significant Additionally, they may casue traffic jams that making eveyone on the road unhappy.

In this project, we will create a data visualization that presents:
  • When does collision occurs more often
  • Where does collision ooccurs more often
  • Which factors contribute to more collisions
  • What types of vehicles involve in more collisions

Data


Data Soucre: Motor Vehicle Collisions - Crashes (NYC Open Data)

link: https://data.cityofnewyork.us/Public-Safety/Motor-Vehicle-Collisions-Crashes/h9gi-nx95

Description: This data set contains details (such as date, time and geography location) of vehicle collisions starting from 7/1/2012. The data, is gathered from all police reported motor vehicle collisions in NYC. Collisions with less than $1000 worth of damage is not required to file a police report and hence may be be included in the data set.

The following columns from the data set are being used for the analysis:
  • CRASH DATE - The occurrence date of the collisions:
    After the data is loaded to our app, the date value is separated into year and month for the analysis
  • CRASH TIME - The occurrence time of the collisions:
    After the data is loaded to our app, the hour value (0-23) is extracted from the time for the analysis
  • LATITUDE - The latitude coordinate of the collisions
  • LONGITUDE - The longtitude coordinate of the collisions
  • CONTRIBUTING FACTOR VEHICLE 1 - The factor contributing to the collision for the primary vehicle
    There are more than 60 unique values for the factors from the raw data. We will convert the original factors into fewer number of revised factors using the manually created table from the following link
    https://github.com/ezaccountz/Data_608/blob/main/Final_Project/factors.csv
  • VEHICLE TYPE CODE 1 - The vehicle type of the primary vehicle
    There are more than 900 unique values for the vehicle types from the raw data. Most of the values are incomplete description of the vehicles. For example, we have the vehicle types "COMER", "F650" and "CAT". We will convert these vehicle types into "Commercial", "Pickup Truck" and "Construction Vehicle". We will convert all of the original vehicle types into fewer number of revised vehicle types using the manually created table from the following link
    https://github.com/ezaccountz/Data_608/blob/main/Final_Project/vehicle_types.csv

User Interface Layout


Factors Drop Down
Year Drop Down Vehicle Types Drop Down
Monthly Distribution Bar Chart Hourly Distribution Bar Chart
Total number of collisions text [All Data] Total number of collisions text [Selected Data]
Collision Scatter Plot Map [All Data] Collision Scatter Plot Map [Selected Data]
Factors Pie Chart [All Data] Vehicle Types Pie Chart [All Data] Factors Pie Chart [Selected Data] Vehicle Type Pie Chart [Selected Data]

***In the following description:
[All Data] indicates the visualization includes all collisions within the view area of the Collision Scatter Plot Maps and it is not filtered by the Factors and Vehicle Types selected in the Factors Drop Down and Vehicle Types Drop Down.
[Selected Data] indicates the visualization includes all collisions within the view area of the Collision Scatter Plot Maps and it is filtered by the Factors and Vehicle Types selected in the Factors Drop Down and Vehicle Types Drop Down.
  • Year Drop Down: single selection, select the data set of which year
  • Factors Drop Down: multiple selection, select the factors of interest. A use may also click on a factor on the Factors Pie Chart [All Data] to select/unselect a factor
  • Vehicle Types Drop Down: multiple selection, select the vehicle types of interest. A use may also click on a vehicle type on the Vehicle Types Pie Chart [All Data] to select/unselect a vehicle type
  • Monthly Distribution Bar Chart: a stacked bar chart the shows the number of collisions for each month through the year, within the view area of Collision Scatter Plot Maps.
    The lower part of the bars shows the number of collisions for Selected Data.
    The whole bars (lower + upper) shows the number of collisions for All Data.
  • Hourly Distribution Bar Chart: a stacked bar chart the shows the number of collisions for each hour through the year, within the view area of Collision Scatter Plot Maps.
    The lower part of the bars shows the number of collisions for Selected Data.
    The whole bars (lower + upper) shows the number of collisions for All Data.
  • Total number of collisions text [All Data]: a text that shows the total number of collisions ploted for All Data
  • Total number of collisions text [Selected Data]: a text that shows the total number of collisions for Selected Data
  • Collision Scatter Plot Map [All Data]: a scatter plot on a map showing the collisions for All Data. A use may move or zoom in/out to select a different view area. The view area of the Collision Scatter Plot Map [Selected Data] will also be updated accordingly
  • Collision Scatter Plot Map [Selected Data]: a scatter plot on a map showing the collisions for Selected Data. A use may move or zoom in/out to select a different view area. The view area of the Collision Scatter Plot Map [All Data] will also be updated accordingly
  • Factors Pie Chart [All Data]: a pie chart that shows the percentages of the factors for All Data. A use may click on a factor to select/unselect the factor from the Factors Drop Down
  • Vehicle Types Pie Chart [All Data]: a pie chart that shows the percentages of the vehicle types for All Data. A use may click on a vehicle type to select/unselect the vehicle type from the Vehicle Types Drop Down
  • Factors Pie Chart [Selected Data]: a pie chart that shows the percentages of the factors for the collisions for Selected Data
  • Vehicle Types Pie Chart [Selected Data]: a pie chart that shows the percentages of the vehicle types for Selected Data
